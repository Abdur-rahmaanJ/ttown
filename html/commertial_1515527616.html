Gas Station Pack by mdk_813 [commercial] UPDATE
Hi guys, I've created a Gas Station Pack that contains four variants of gas stations from different oil-companies. The gas stations are 3x3 level-1 commercial buildings with 40 workers. The buildings were designed by me, but the cars used on the lots are some of the original cars from Theotown made by Lobby and theotheoderich. The tarmac texture was also created by me, but it is designed to align with the commercial parking lot pieces in the decorations menu. In sandbox mode the gas station appears as one entry with four options to choose. In simulation mode the four variants should build randomly. If any problems occur, especially in simulation mode, please let me know! UPDATE: I realized that the scale of the building was a bit of. As people are 3 pixels tall in Theotown, the windows and roofs of the gas stations were too high. So I lowered them a bit. This is really only a minor change and if you are not bothered by the original scale, then there is no real need to download the new version. But if you do want to update the plugin, then you only need to replace the old pngs in your plugin folder with the new ones provided in this update, because the json was not changed.
