Sunken Highway Walls mdk_813 [Deco]
Hi my fellow Theonians, this one might be one of my last plugins for an unforeseeable time, as I will have much less time for personal stuff in the coming months due to professional reasons. But in the meantime, I would like to finally share my Sunken Highway Walls, which some of you might have seen already on a screenshot in another thread some weeks ago. I've been pondering over this plugin for a long time now, considering to change this and that, but now I realized, that most likely I won't do anything more to it. So I thought, I might as well release it now, before forgetting all about it. The walls are probably not perfect, but some of you might find them useful anyway. The plugin can create the illusion of a layered surface. It works different to Kingtut's Elevated Terrain in that it is purely fake-3D. It does not actually add a an additional level of terrain, it just looks that way. You can use it to create the appearance of both a higher and a lower level (see images below). The set contains five different rotation aware puzzle pieces, that you will have to place one by one. (Sorry, a linetool is not possible here. Believe me, I've tried it...): - a straight piece - an outer corner - an inner corner - a right end - a left end I recommend using the end-pieces next to crossing bridges to make a believable transition, as underbridge-pieces are not possible, as far as I know. The pieces have a minimal (monthly) price, but of course you can set these to 0 in the json if you choose so. Of course, You can use these walls with any street-type, not only with highways, or even without any road at all. Have fun!

Flag Pack + Templates mdk_813 [deco]
Hi folks, as one of the most popular plugins for theotown, the flags by "Sometheotown mayor", won't be developed any further, I decided to release my personal flag plugin which I made quite some month ago. It was initially part of my Modular Park Set but I've never released it so far (Look at the screenshot of the Modular Park and you'll see it! It's funny that nobody ever noticed it ). The set contains numerous flags of different countries that use "draw ground". That means, the game’s ground texture will be visible. If you want to place the flag on another texture, I recommend you use Josh’s “ground” decals, because they will allow you to place different textures beneath these flags inside the game. Here is a link to @Josh ’s decals: viewtopic.php?f=76&t=4654 The foundation of the flagpole is taken from the game and was made by Lobby & Theo. Please note, that I won't take any requests to make additional flags, because I just don't have any time for that! However, below I'll provide you with templates (PSD and TIF as well as the JSON) to create your own flags. You can use free applications like GIMP or Paint.net (plugins may be necessary) to open and edit these template images. The JSON can be edited with Wordpad on Windows for example. I’m sure, there are options to edit these files on Android as well (if you know some apps, please write a comment below!) How to use these templates: The template images contain several layers (listed here from bottom to top): the flagpole three layers for each flag size (6px, 7px and 8px. Pick the one you need according to the complexity of your flag!): an example layer . a bright green layer reserved for your personal design (paint here! You only need the base-colors!). a layer for lights and shadows (Don’t touch this!). Remember: Only work on the bright green layers! Pick the colors you need and paint over the green flag. Note that the flag is bend due to the wind so that your flag design needs to have some steps in it (Consult the example layers to see what I mean) Save your finished flag as a PNG-file. Use the naming structure that I have provided and replace “YOURCOUNTRY” and “USERNAME” according to your flag. Inside the json, you also have to replace "YOURCOUNTRY" and "USERNAME" the same way. Make sure the file name specified in "frames" is exactly the same as the name of the image which you created before! Then, save the json and again replace "YOURCOUNTRY" and "USERNAME" accordingly. Finally, put your PNG and JSON into a folder, which you give the same name as the JSON (without the file-extension). Then pack this folder to a zip-file and transfer it to your device. You are free to share your creations in this thread below as zip-files (and nowhere else!). It is important that you adhere to the provided naming convention, in order to avoid conflicts with the flags of other users. Please add a screenshot from the game so that people can see what they're downloading! Have fun!

Traffic Barriers mdk_813 [deco]
Hi, this time I would like to present a selection of traffic barriers which you can use for your highways, but also avenues and other road types. It consists of three separate plugins so that you can select the parts you want to use and leave out those which you won't need. Here are the three parts: traffic_barrier_median_Linetool_mdk_813.zip (18.18 KiB) Downloaded 715 times This is probably the basic and most important part, which works fine on its own, i.e. even without the other two parts. It adds median barriers which are intended to be placed between the lanes of a highway. It consists of two linetools, one with lamps, one without lamps. Personally I think the spacing between the lamps in the linetool is too narrow, but I included it anyway, because some people might feel different about it. But I also added a rotation aware puzzle piece with a lamp that allows you to create variable spacings between the lamps. To use this you would want to draw a barrier with the linetool without lamps first and then put the single lamp piece over it whereever you want a lamp to be. I suggest to place on every second tile (see example picture above). As long as you don't draw over it with the linetool again, the barriers will retain a correct transition to the puzzle pieces. traffic_barrier_single_mdk_813.zip (26.61 KiB) Downloaded 534 times This is an optional part. It adds puzzle pieces of single barriers which are intended to be placed at the sides of the lanes. I wanted to make these as linetools, too, but unfortunately map rotation wouldn't work due to the asymmetry of the tiles. So I had to stick to rotation aware puzzle pieces. Note that outer corners are not included because they are unnecessary. Just leave the outer corner tiles empty. traffic_barrier_single_Soundwall_mdk_813.zip (43.88 KiB) Downloaded 538 times This is another optional part. It acts as an addon to my New Soundwalls plugin: viewtopic.php?f=70&t=4279 Essentially, these pieces are the same as the above single barriers just with Soundwalls added to them. You can use these pieces in conjunction with my New Soundwalls in places where you want to have a barrier at the side of the roads. Again, a linetool is not possible here unfortunately. These pieces have the same noise reductive function as the New Soundwalls. Please note, that here you would have to place the provided outer corner (OC) tiles. Hi folks, today I have some New Soundwalls for you which are not puzzle pieces but a linetool that uses "frame alignment" and works just as the road tools do. It is a complete rework of my first plugin for Theotown, which back then I could not have done without the help of Dusko who kindly provided a json file. You can find the old Soundwalls here: viewtopic.php?f=63&t=3394 Here you can follow the initial development of the old plugin: viewtopic.php?f=34&t=3384 There are a number of reasons for this rework: - I wanted to turn the Soundwalls into a linetool to make their placement much easier. - The Soundwalls have reworked textures so that they look less plain now. - The shadows have been reworked so that the "dent" from the shadows of the original walls is eliminated. - The new Soundwalls now have the ability to form X- and T-intersections. - I plan on including the Walls in a future plugin, but unfortunaley Dusko has been unreachable for quite some time now, so it became necessary to create a whole new plugin. As I've said, the New Soundwalls are a linetool which you can use to draw walls the same way as you would draw a road. It still has a noise reductive function but it is now part of the deco menu (whereas the old walls were in the park menu). Have fun! P.S. Dusko, if you read this, we miss you here!
