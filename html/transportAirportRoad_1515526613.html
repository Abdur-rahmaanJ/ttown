It is intended as a quick commuter train for local or regional transportation. This plugin contains: - railways (a normal version and a slower version for stations) with bridge frames. - functional stations in two variants (one to be placed at the side of a single railway, another to be placed in between two railways) - non-functional deco tiles to extend the station platforms. - a German-style S-Bahn. - a Gravel Decal to extend the railway areas. I recommend using bridges to cross normal roads in order to avoid unwanted effects. The plugin requires the inter-station category by KINGTUT101 to work. You can download it from here: http://www.theotown.com/forum/viewtopic ... 364#p72364 Put the downloaded category folder into your plugin folder! Make sure the categories' folder name starts with a 1, so that it is read in by the game before the S-Bahn plugin! NOTE: If you have already used the earlier version of the platforms for decoration purposes, delete them in your cities and from your plugin folder before installing this plugin!
